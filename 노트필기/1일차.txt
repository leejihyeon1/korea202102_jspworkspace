1.java SE 
  - 독립실행형 응용프로그래밍
    프로그램의 실행 코드가  해당 pc에 있어야 하는 방식 
  - java 개발에 사용되는 개발 키트 jdk (jvm, jar..)

2.javaEE( 대형 어플리케이션 == 기업시장)
 - javaSE을 WEB으로 확장한 기술
 - javaEE기반의 프로그램을 개발하기 위해 oracle에서 별도로
   설치할 프로그램은 없다!!! 즉 서버만 준비하면 된다!!
   이 서버는 오라클에서 독점하지 않고, 서버 시장은 오픈 되어 있다!!
   즉, 서버를 개발할 수 있는 사업자라면 누구라도 , javaEE기반의 웹서버를
   제작 및 배포 , 판매할 수 있다..
   - 웹컨테이너 ( EJB 실행 불가)
      유료 : Resin
      무료 : Tomcat (apache)
   - WAS(Web Application Server) - EJB 지원 
      JEUS, WebLogic ( Bea사를 Oracle이 인수), JBoss(Red Hat)...

   - EJB란 ?  스프링 기술이 출시되기 전에 자바의 웹시장에서 사용된 기업용
